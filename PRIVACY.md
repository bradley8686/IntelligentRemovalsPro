# Privacy Notice

Intelligent Removals Pro uses the device camera only while the user starts an AI survey. Video frames are processed locally in the app window for object detection. The app does not upload camera footage to Intelligent Removals Pro servers.

Survey exports are created only when the user chooses JSON or CSV export. Exported files may contain item names, quantities, estimated weights, estimated volumes, source labels, confidence values, timestamps, and vehicle recommendations.

Operators should store exported survey files according to their own customer data policy and delete files that are no longer required.

The current AI model is loaded from public TensorFlow CDN assets. For fully offline deployments, package the model assets with the app before release.
